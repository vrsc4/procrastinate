# ClockGain VPN Setup

This document explains how to set up and run the ClockGain VPN server.

## Prerequisites

- Node.js (automatically installed by setup)
- An internet connection

## Installation

1. Double-click `node-installer.msi` to install Node.js if it hasn't been installed automatically
2. After Node.js installation is complete, double-click `setup.bat` to install dependencies and start the server
3. The server will start on port 3000

## Using the VPN

1. Open ClockGain in your browser
2. Go to the VPN page
3. Add the websites you want to block
4. Set the duration
5. Click "Connect VPN"

## Features

- Blocks specified websites for all devices on your network
- Custom block page with a focus-friendly message
- Automatic timeout based on specified duration
- Works across all devices that access the website

## Troubleshooting

If you encounter any issues:
1. Make sure Node.js is properly installed
2. Try running `setup.bat` again
3. Check if port 3000 is available
4. Make sure you have administrator privileges
